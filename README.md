# Door James 🚪

An internal meeting room booking demo application built using Spring Boot.

## Features
- Authentication (admin/admin123)
- Book meeting rooms
- Generate meeting links
- Send email notifications (Gmail SMTP)
- Simple UI

## Tech Stack
- Java (Spring Boot)
- Thymeleaf
- Gmail SMTP

## Run Locally

1. Clone repo
2. Open in IntelliJ
3. Update application.properties with Gmail credentials
4. Run DoorJamesApplication

Visit:
http://localhost:8080
